Einführung HTML und CSS

HTML

HTML (HyperText Markup Language) ist die Auszeichnungssprache, die die Struktur und den Inhalt einer Webseite definiert, indem sie Elemente wie Überschriften, Absätze, Bilder und Links festlegt.

CSS

CSS (Cascading Style Sheets) ist die Gestaltungssprache, die das visuelle Erscheinungsbild einer Webseite steuert, indem sie Stile wie Farben, Schriftarten, Abstände und Layouts für die HTML-Elemente definiert.
